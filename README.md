# zxz

Dead simple JavaScript, ESM, and TypeScript scripting

## Installation

```bash
pnpm add -g zxz
```

```bash
yarn global add zxz
```

```bash
npm install -g zxz
```

## Usage

### CLI Usage


```bash
zxz [script] [options]
```
